# Changelog

## 15 июля 2026 — Создание проекта «Стиль & Релакс»

- Создан новый одностраничный лендинг в папке `prg1/`
- Название бренда: «Стиль & Релакс» — сервис онлайн-записи к парикмахеру и массажисту
- Реализованы все 9 секций: шапка, hero, преимущества, услуги, мастера, форма записи, отзывы, контакты, футер
- Чистый HTML, CSS и JavaScript (без фреймворков)
- Адаптивный дизайн (mobile-first) от 320px
- Единая цветовая гамма: тёмно-зелёный (#1E3A3F), белый, золотой акцент (#C9A96E)
- Шрифты: Playfair Display (заголовки) и Inter (текст) через Google Fonts
- Плавный скролл по якорным ссылкам
- Анимация появления блоков при скролле (Intersection Observer)
- Кастомный JS-календарь без сторонних библиотек
- Маска ввода телефона (+7)
- Валидация всех полей формы на стороне клиента
- Табы для переключения между парикмахерскими услугами и массажем
- Слайдер отзывов с автоматическим перелистыванием
- Динамическое обновление списка услуг при выборе типа
- Предвыбор мастера при переходе с карточки
- Кнопка «Наверх» с плавной прокруткой
- Семантическая вёрстка, SEO-оптимизация (meta-теги, alt)
- Гамбургер-меню для мобильных устройств
- Стили вынесены в `css/style.css`, скрипты в `js/script.js`
