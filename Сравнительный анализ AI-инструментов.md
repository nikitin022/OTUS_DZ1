## 0. Анализ требований проекта

- **Создание сайта регистрации на услуги парикмахера и массажиста.**

- **Требования:**
  1. Сложность — средняя.
  2. Необходимость доступа к локальным файлам для дальнейшей работы над проектом и возможности выгрузить его на свой хостинг.
  3. Необходимость обезопасить данные проекта.
  4. Бюджет минимальный или нулевой.

### 🏆 Рекомендуемый инструмент: **Cline**

**Аргументы:**

1. **Локальные файлы** — расширение для VS Code, прямой доступ к файловой системе проекта (**4/5**).
2. **Безопасность** — полная обработка на локальной машине, можно подключать локальные модели (**5/5**).
3. **Бюджет** — бесплатный, оплата только за API-токены (**5/5**).
4. **Сложность** — справляется с проектами средней сложности, код чистый и структурированный (**4/5**).
5. **Выгрузка на хостинг** — проект остаётся в локальной папке, любой файл можно выгрузить на хостинг.

### 🥈 Альтернатива: **Aider**

Если Cline не подойдёт, **Aider** — отличная замена:
- Бесплатный, open-source.
- Прямой доступ к файловой системе.
- Каждое изменение через Git — удобная история версий.
- Код работает без правок в 71% случаев.

### ❌ Почему не подошли альтернативные варианты

#### Облачные агенты

| Инструмент | Почему не подошёл |
|---|---|
| **Bolt.new** | ❌ **Облачная среда** — код обрабатывается на удалённых серверах, нет прямого доступа к локальным файлам. ❌ **Безопасность** — данные проекта покидают локальную машину. ❌ **Бюджет** — при масштабировании стоимость может вырасти до $25/мес. |
| **Lovable** | ❌ **Облачная среда** — ограниченная работа с локальной файловой системой. ❌ **Безопасность** — проект хранится в облаке, что не соответствует требованию по защите данных. |
| **Replit Agent** | ❌ **Облачная среда** — вся разработка ведётся в среде Replit, нет прямого доступа к локальной файловой системе. ❌ **Бюджет** — при активном использовании может потребоваться платный тариф до $100/мес. |

| **GitHub Copilot** | ❌ **Бюджет** — $10/мес, хотя и недорого, но не полностью бесплатно. ⚠️ **Безопасность** — код сканируется облачным сервисом, что может быть неприемлемо для конфиденциальных данных. |
